Overview

This application has four web services: assign4, topsis, entropy and critic web services

assign 4 web service

Method and URI Functionality
GET | http://localhost/assign4/{filename} parses JSON data file and inserts the decision matrix into a database (e.g., MySQL) that is stored on a remote web location such as GCP Cloud SQL. This files becomes then the default data file to be used for the below functionalities
POST | http://localhost/assign4/add inserts a record into decision matrix stored in a database (e.g. MySQL)
PUT | http://localhost/assign4/update/{alternative_name} updates a record in decision matrix stored in a database (e.g. MySQL);
DELETE | http://localhost/assign4/delete/{alternative_name} deletes a record in decision matrix stored in the database (e.g., MySQL)
DELETE | http://localhost/assign4/delete deletes all records in decision matrix stored in the database (e.g., MySQL)
GET | http://localhost/assign4/process returns the results in JSON format containing: 1. TOPSIS with Entropy: alternative names, rank and performance scores, 2. TOPSIS with CRITIC: alternative names, rank and performance scores, 3. Correlation coefficient comparing the performance scores of TOPSIS with Entropy and TOPSIS with CRITIC, respectively
GET | http://localhost/assign4/process/entropy eturns the results in JSON format containing the alternative names, rank and performance scores after computing TOPSIS using entropy weight estimation
GET | http://localhost/assign4/process/critic returns the results in JSON format containing the alternative names, rank and performance scores after computing TOPSIS using critic weight estimation
GET | http://localhost/assign4/process/correlation returns the results in JSON format containing the correlation coefficient comparing the performance scores of TOPSIS with entropy and TOPSIS with CRITIC methods, respectively.

critic web service

Method and URI Functionality

topsis web service

Method and URI Functionality
GET | http://localhost/assign4/topsis computes the decision matrix using TOPSIS with using (a) entropy and (b) critic weight estimation. This service returns the results in JSON format containing: 1. TOPSIS with Entropy: alternative names, rank and performance scores 2. TOPSIS with CRITIC: alternative names, rank and performance scores 3. Correlation coefficient comparing the performance scores of TOPSIS with Entropy and TOPSIS with CRITIC, respectively
GET | http://localhost/assign4/topsis/entropy computes the decision matrix using TOPSIS with using the entropy weight estimation method. This service returns the results in JSON format containing the alternative names, rank and performance scores
GET | http://localhost/assign4/topsis/critic computes the decision matrix using TOPSIS with using the critic weight estimation method. This service returns the results in JSON format containing the alternative names, rank and performance scores
GET | http://localhost/assign4/topsis/correlaton computes the correlation coefficient between TOPSIS using entropy method and TOPSIS using the critic method and returns the results in JSON format.
